EJERCICIO 1
crear una lista de contactos el usuario puede eliminar y crear contactos
requisitos: 
crear una interfaz contacto con el id, nombre, teléfono, email 
el componente debe de tener un formulario con 3 input:
nombre, teléfono, email
un botón para agregar el contacto 
una lista que muestre todos los contactos 
un boton para eliminar un contacto

las funcionalidades deben de ser: 
- agregar un nuevo contacto (generar un ID automático con alguna librería npm)
- eliminar un contacto por su ID
- Limpiar el formulario después de agregar
- mostrar un mensaje si no hay contactos almacenados
